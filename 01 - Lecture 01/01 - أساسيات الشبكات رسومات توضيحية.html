📡 CCNA المتكامل | رسومات توضيحية وشروحات احترافية

🧠 مناهج CCNA مع رسوم بيانية (SVG) — شرح أجهزة الشبكة، الطبقات، العناوين، Protocols

🔧 الأدوات الأساسية (Voice & Data)

📡 رسم توضيحي: Splitter و Filter و RJ45

Splitter: يقسم الحرارة إلى Voice & Data. Filter: تنقية الصوت. Rosetta (RJ11) لسلك التليفون. RJ45 (Connector) لسلك الإنترنت.

NIC (Network Interface Card): كارت تعريف الشبكة. MAC Address: عنوان جهازي فريد.

🏠 LAN

⚙️ أجهزة الشبكة الأساسية (HUB, Switch, Router, Modem)

✔ HUB: 16 مخرج، إرسال لجميع المنافذ. ✔ Switch: 48 مخرج، جدول MAC ذكي. ✔ Router: توجيه بين الشبكات. ✔ Modem: تحويل الإشارة التناظرية والرقمية.

📟 IP Address (IPv4 & IPv6) + Classful Addressing

IPv4: 4 octets, IPv6: 8 octets. Subnet mask تحدد أجزاء الشبكة. Host: حتى 255 جهاز.

🧠 بروتوكولات حيوية (DHCP, APIPA, NAT, PAT, DNS)

DHCP في المودم، APIPA حل بديل (169.254.x.x). NAT/PAT يسمح بخروج الأجهزة للإنترنت. DNS يحول أسماء النطاقات. VOIP للاتصال عبر الأقمار الصناعية.

📚 نموذج OSI - الطبقات السبع (7 Layers)

TCP vs UDP: TCP يضمن وصول البيانات، UDP إرسال دون تأكيد. HTTP/HTTPS للتصفح الآمن وغير الآمن. FTP رفع وتحميل، SMTP/POP3 للإيميلات.

🛠️ حلول مشكلات الإنترنت والتشخيص بالأوامر

# أوامر CCNA الأساسية
ipconfig /flushdns ← مسح ذاكرة DNS cache
tracert google.com ← تتبع مسار الحزمة وتحديد العطل
netstat -a ← كشف spyware (Established أكثر من 4 مرات)
nslookup cisco.com ← معرفة IP الموقع
ping 8.8.8.8 -t ← اختبار الاتصال المستمر
✔ حل مشكلة DSL: تغيير مكان السلك، إبعاد المودم عن المجالات الكهربائية، فحص الـ Splitter. ✔ عند وجود X حمراء نتبع 6 خطوات (تغيير port، كابل النت، كارت الشبكة enable/disable).

📶 Wireless وشبكات الواي فاي + أجهزة التقوية

مشاكل وحلول: التأكد من زر الواي فاي، إظهار الشبكة المخفية، استخدام Access Point للمسافات البعيدة، Repeater للمسافات القريبة. Proxy لفتح المواقع المحجوبة، Fairwall حماية.

✨ مفاهيم داعمة متقدمة (CCNA Plus)

🔄 CSMA/CD (Carrier Sense Multiple Access / Collision Detection): تستخدم في Ethernet لتجنب التصادمات في شبكات HUB. 🌐 Broadcast Domain vs Collision Domain: Switch يقسم domains collision، بينما Router يفصل broadcast domains.

📌 VLAN (Virtual LAN): تقسيم الشبكة منطقياً لزيادة الأمان وتقليل البث. 🔐 SSH vs Telnet: Telnet غير آمن، SSH تشفير كامل للجلسات عن بعد.

⚡ Static Routing vs Dynamic Routing: Static يدوي، Dynamic باستخدام بروتوكولات مثل OSPF / RIP / EIGRP.

🖧 IPv6 اختصار للعنوان: 8 مجموعات هكساديسيمال، تحسين في الأمان ووفرة العناوين.
